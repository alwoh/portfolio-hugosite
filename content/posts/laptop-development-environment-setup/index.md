{{< blogparagraph "I recently got a new laptop back in January ‘26.  No, it's not a MacBook.  No, it doesn't have Windows as its operating system.  And no, I haven’t picked a flavored Linux distribution.  You can probably deduce from this that it’s a Chromebook. But why?" "">}}

{{< blogparagraph "Well, first off, cost.  I had found it on eBay for a good price.  Two, it is actually quite fast considering it has only 8 GB of RAM.  And three, it’s pretty seamless with all the tools I have used on my tech journey." "">}}

{{< blogparagraph "Currently, I have a somewhat out-of-the-box setup for my development environment.   What I wanted was the ability to do most of my work within the terminal.  So I was able to go through the ChromeOS settings and turn on the Linux development environment." "">}}

{{< blogparagraph "Then, to set up all my development software installed a couple of things to make it easier to get around without much use of a touchpad or mouse." "">}}

{{< blogparagraph "First, tmux, a terminal multiplexer, allows me to open multiple terminal sessions." "">}}

{{< blogparagraph "Second, instead of using bash, I’ve installed fish for its easy autocomplete and great help manual." "">}}

{{< blogparagraph "And finally, the Helix editor, while I have used Vim in the past, I found Helix to be great with easy menus to help, and not much configuration needed." "">}}
